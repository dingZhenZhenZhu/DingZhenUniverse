# 丁真宇宙 DingZhenUniverse 
本仓库致力于收集丁真珍珠先生的赞美MEME, 讲好丁真故事, 传播丁真声音
## 丁真语法
丁真语法起源于"一眼丁真", 其基础拼音语法为: "yī yǎn dīng zhēn", 以此拼音为基础衍生出了丁真语法范式: "y? y? dīng zhēn", 其中以一三声构成最为经典.
